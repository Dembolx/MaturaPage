📚 Teoria i złożoność obliczeniowa

⏱️ Notacja O – złożoność

O(1) – stała (dostęp przez indeks)
O(log n) – logarytmiczna (wyszuk. binarne)
O(n) – liniowa (przeszukiwanie liniowe)
O(n log n) – merge sort, quicksort śr.
O(n²) – bubble/insertion/selection sort
O(2ⁿ) – wykładnicza (generowanie podzbiorów)

🔀 Algorytmy sortowania

Bubble sort: O(n²) – porównuj sąsiednie, zamieniaj
Merge sort: O(n log n) – dziel i scalaj (stabilny)
Quick sort: O(n log n) śr., O(n²) najg. – pivot
Insertion sort: O(n²), O(n) dla prawie posortowanych
Heap sort: O(n log n) zawsze, in-place

🗄️ Podstawy SQL

SELECT kolumna FROM tabela WHERE warunek;
SELECT * FROM t ORDER BY kol DESC;
SELECT k, COUNT(*) FROM t GROUP BY k;
SELECT a.* FROM a JOIN b ON a.id=b.aid;
-- HAVING filtruje grupy (po GROUP BY)
